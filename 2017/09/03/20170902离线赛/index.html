数学 dp 可持久化并查集 启发式合并

A、猜数——UOJ#12

关键字：数学

Task：

这一天，小Y、小 D、小C 正在愉快地玩耍。

小Y 是个数学家，他一拍脑袋冒出了一个神奇的完全平方数 n。

小D是个机灵鬼，很快从小Y嘴里套出了 n 的值。然后在脑内把 n 写成了 $a\times b$的形式。

其中 a,b 都是正整数。

小C是个八卦狂，他发现小D从小Y那里获知了神奇的东西，于是死缠烂打追问小D。

最后小D 说道：“我可以告诉你正整数 g 和 l 的值，我保证 $a\times b=g\times l=n$ 且 a,b 都是 g的倍数。但是 a,b 我可不能告诉你。”

这可急坏了小C。他决定退而求其次，找出$a+b$的最小值和最大值。请你帮帮他吧！

【分析】

一开始开题目其实有点慌，第一题就是数学？

$a=a’\times g$

$b=b’\times g$

$n=a’\times b’\times g^2$

又因为$n=g\times l$

所以$a’\times b’=l/g$

根据平均数原理，a+b的最大值当a与b相差最大时取，最小值则是最接近接近的时候。

又因为n为完全平方数，所以$a’\times b’$为完全平方数，开方即可。

【反思】

有些人可能会把g和l乘起来再开方，但是开方返回的类型是double，所以会因为精度问题挂掉。所以，再强调一遍，double的有效精度只有16位！

B、任务

关键字：dp

Task：

现在又两台机器 A 和 B。有 n 个任务，编号 1,2,….,n。你必须把每个任务安排到一台机器上处理，同时需要满足以下一些条件。

（1）你必须把每个任务安排到任意一台机器上处理。

（2）在任何时刻，一台机器只能最多处理一个人任务。

（3）任务 $i(0 < i < n)$可以被处理当前仅当每个任务 $j（ j< i）$已经被完成或者正在进行。

（4）一个任务如果在一台机器上进行，它是不能被打断的。

请你算算最少完成任务的时间。

【分析】

显然是$dp$。

我们只关心在做某个任务时，第一台和第二台机器什么停止工作。

但是如果把两个时间都存在$dp$下标上的话，只能过30%的数据。那么我们只在$dp$下标上存一台机器的时间，另一台机器停止工作的最短时间为$dp$的值，这样就可以过70%了。

但是这个$dp$状态好像并不能优化到100分……

正解的$dp$也是从一种70分的做法优化过来的：定义$dp[i][j][k]$表示做到第i个任务时，第一台机器还有j时间做完，第二台机器还有k时间的最小时间花费。

至于正解，则将两个时间的时间差放在$dp$状态下。为什么能够这样做呢？因为题目的（3）条件，若时间差大于3000（每个任务花费的最长时间），那么一定需要等待截止时间短到截止时间长的开始，才能进行下一个任务。这样的话，两个机器的截止时间的差一定小于等于3000，也就能使复杂度到达可以接受的地步了。

【反思】

这道题正解的70分应该算是比较正常的$dp$定义了。而且从70分优化到100分的将两个时间做差放在$dp$状态上也是比较常见的优化方法。

C、图操作——UOJ#14

关键字：可持久化并查集、启发式合并

Task：

开始有 n 个点,现在对这 n 个点进行了 m 次操作,对于第 i 个操作(从 1 开始编号)有可能的三种情况:

1、Add a b: 表示在 a 与 b 之间连了一条长度为 i 的边(注意, i是操作编号)。保证 1≤a,b≤n。

2、Delete k: 表示删除了当前图中边权最大的k条边。保证 k 一定不会比当前图中边的条数多。

3、Return: 表示撤销第 i−1 次操作。保证第 1 次操作不是 Return 且第 i−1次不是 Return 操作。

请你在每次操作后告诉$DZY$当前图的最小生成树边权和。如果最小生成树不存在则输出 0。

【分析】

30%的数据模拟即可。

50%因为边权是递增的，所以只要开始求出最小生成树的话，那么接下来的答案是不会发生改变的，但比赛的时候忘记了边权是递增的这一条件，便将题目转化为我们做过的“删边最小生成树”，但是这与之前的题目又不同，那个只需要求出删除一条边后的最小生成树。所以就没写了……

70%没有Return操作。添加操作使得$sz$小的集合并到大集合，然后暴力添加即可。不过添加时需要额外的一个信息：以前该点所在集合的id。删除操作则可以用之前额外的那个信息时信息复原。因为添加的时候是启发式合并，删除时每条边最多只会删除一次，所以复杂度是有保证的。

至于100%，多出了一个Return操作，对于“Add Return”组合，删除即可。但是如果是“Delete Return”组合，我们可以事先存下使用当前权值前k小的边时最小生成树大小直接输出即可。

【反思】

50分没写到是真的不应该。看来小C说的在看题的时候将注意事项写在$txt$里要认真做。
